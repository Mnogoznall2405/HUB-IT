# IT-invent Bot

Telegram-бот для учёта IT-оборудования, поиска по инвентарной базе и регистрации сервисных работ.

## Что умеет бот

- Поиск оборудования по серийному номеру, фото и сотруднику
- Просмотр оборудования по типу, филиалу и базе данных
- Регистрация перемещений с формированием акта
- Регистрация работ:
  - замена батареи ИБП
  - замена компонентов ПК
  - чистка ПК
- Экспорт:
  - ненайденного оборудования
  - перемещений
  - замен батареи ИБП
  - чистки ПК
  - замен компонентов ПК
  - полного содержимого выбранной базы

## Что убрано

Из пользовательского интерфейса полностью снят поток замены комплектующих МФУ и связанный экспорт. Исторический файл `data/cartridge_replacements.json` сохраняется как архив, но бот больше не создаёт новые записи этого типа и не показывает этот сценарий в меню.

## Запуск

Из корня проекта:

```bash
python -m bot.main
```

## Основные данные

- `data/unfound_equipment.json` — ненайденное оборудование
- `data/equipment_transfers.json` — история перемещений
- `data/battery_replacements.json` — замены батареи ИБП
- `data/pc_cleanings.json` — чистки ПК
- `data/component_replacements.json` — замены компонентов ПК
- `data/cartridge_replacements.json` — исторический архив снятого МФУ-потока
