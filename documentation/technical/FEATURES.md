# Описание функций IT-invent Bot

## 1. Поиск оборудования

- Поиск по серийному номеру
- OCR по фотографии
- Поиск по сотруднику с подсказками
- Просмотр по типу оборудования и филиалу

## 2. Перемещение оборудования

- Распознавание серийных номеров по нескольким фото
- Формирование акта приёма-передачи
- Отправка результата в чат или по email

## 3. Управление базами данных

- Выбор рабочей базы
- Просмотр статистики
- Экспорт выбранной базы

## 4. Экспорт данных

- Ненайденное оборудование
- Перемещения оборудования
- Замены батареи ИБП
- Чистка ПК
- Замены компонентов ПК

## 5. Регистрация работ

### 5.1 Замена батареи ИБП

- OCR или ручной ввод серийного номера
- Поиск оборудования в SQL
- Подтверждение карточки перед сохранением
- Сохранение в `data/battery_replacements.json`

### 5.2 Замена компонентов ПК

- Поиск ПК по серийному номеру
- Выбор компонента из списка
- Сохранение в `data/component_replacements.json`

### 5.3 Чистка ПК

- Поиск ПК по серийному номеру
- Отображение истории чисток
- Сохранение в `data/pc_cleanings.json`

## 6. Архив снятого функционала

- `data/cartridge_replacements.json` сохраняется для истории
- UI и export для замены комплектующих МФУ удалены
- новые записи по МФУ через бот больше не создаются
