# Работы и обслуживание

## Поддерживаемые сценарии

В разделе `Работы` доступны только три сценария:

- замена батареи ИБП
- замена компонентов ПК
- чистка ПК

Сценарий замены комплектующих МФУ из бота убран. Исторические записи по нему сохраняются как архив, но новые записи через бот больше не создаются.

## Замена батареи ИБП

### Как работает

1. Откройте `/start → Работы → Замена батареи ИБП`
2. Отправьте фото серийного номера ИБП или введите его вручную
3. Бот найдёт оборудование в базе
4. Проверьте карточку подтверждения
5. Сохраните запись

### Что сохраняется

- серийный номер
- модель
- филиал
- локация
- сотрудник
- база данных
- метка времени

### Экспорт

Доступен отдельный экспорт замен батареи ИБП по периодам.

## Замена компонентов ПК

### Как работает

1. Откройте `/start → Работы → Замена компонентов ПК`
2. Передайте серийный номер или фото ПК
3. Подтвердите найденное оборудование
4. Выберите компонент из списка
5. Подтвердите сохранение

### Что сохраняется

Запись о замене компонента ПК сохраняется в историю работ и участвует в отдельном экспорте по компонентам ПК.

## Чистка ПК

### Как работает

1. Откройте `/start → Работы → Чистка ПК`
2. Отправьте серийный номер или фото
3. Проверьте найденное оборудование
4. Сохраните запись о выполненной чистке

### Что показывает бот

- дату последней чистки
- количество предыдущих чисток
- относительное время с прошлого обслуживания

### Экспорт

Доступен отдельный экспорт чисток ПК по периодам.

## Архив МФУ-данных

Исторический файл `data/cartridge_replacements.json` не удаляется. Он сохраняется только как архив старых данных и не участвует в текущих пользовательских сценариях бота.
