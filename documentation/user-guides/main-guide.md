# Основное руководство по IT-invent Bot

## Назначение

IT-invent Bot помогает искать оборудование в инвентарной базе, оформлять перемещения и регистрировать сервисные работы по поддерживаемым сценариям.

## Основные сценарии

### Поиск оборудования

- По серийному номеру вручную
- По фотографии серийного номера
- По сотруднику с подсказками
- По типу оборудования и филиалу

### Перемещение оборудования

- Загрузка до 10 фотографий
- OCR-распознавание серийных номеров
- Выбор нового сотрудника
- Формирование акта приёма-передачи
- Отправка результата в чат или по email

### Работы и обслуживание

- Замена батареи ИБП
- Замена компонентов ПК
- Чистка ПК

### Экспорт

- Ненайденное оборудование
- Перемещения
- Замены батареи ИБП
- Чистка ПК
- Замены компонентов ПК
- Полный экспорт выбранной базы данных

## Главное меню

После `/start` пользователь работает через разделы:

- поиск оборудования
- перемещение оборудования
- работы
- экспорт
- базы данных

## Команды

- `/start` — открыть главное меню
- `/help` — показать справку
- `/cancel` — отменить текущий сценарий

## Что больше недоступно

В боте больше нет сценария замены комплектующих МФУ и экспорта по этим данным. История прежних записей остаётся только как архив и не используется в текущем интерфейсе.
